# 🎓 Student Registration System

A modern and responsive web application to manage student records efficiently.  
Built using React and Vite with a clean UI and real-time updates.

---

## 🚀 Features

- Add student details (Name, Roll Number, Course, Gender)
- Search students instantly
- View student list dynamically
- Clear all records
- Clean and responsive UI
- Real-time updates

---

## 🛠 Tech Stack

- React (Vite)
- JavaScript (ES6+)
